CHANGE MY VIEW

This was our first project as a team and one of our favorites to this day We were inspired by the subreddit Change My View - "A place to post an opinion you accept may be flawed" We decided to build a visual reader where you can search any topic and also find hot, new and top posts

Technologies used

  • React frontend hitting Reddit public API
  • Utilized Redux for app state management
  • Responsive mobile first design utilizing Material UI components
  • Logo & Branding creative direction
